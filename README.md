Android MVP
===========

Sample project that shows how to implement MVP in Android and also combines some libraries to handle Dependency Injection, HTTP Requests and View Binding.

Libraries
---------

- Dagger 2.7
- Retrofit 2.1.0
- RxAndroid 1.2.1
- RxJava 1.1.9
- Butter Knife 8.4.0
- Picasso 2.5.2
